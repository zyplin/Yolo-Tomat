📱 Tentang Aplikasi Deteksi Kematangan Tomat

🎯 Tujuan Pengembangan

Aplikasi ini dikembangkan untuk membantu petani, pedagang, dan konsumen dalam mengidentifikasi tingkat kematangan tomat secara cepat, objektif, dan otomatis. Dengan bantuan kecerdasan buatan, pengguna dapat mengambil keputusan yang lebih tepat terkait waktu panen, distribusi, hingga konsumsi.

Deteksi dilakukan berdasarkan citra visual tomat menggunakan kamera atau gambar yang diunggah. Setiap hasil klasifikasi disertai dengan penjelasan deskriptif dan dapat diunduh dalam bentuk CSV.

🧠 Teknologi yang Digunakan

  • 🔍 YOLOv8 – Model deteksi objek real-time berbasis deep learning
  • ⚙️ OpenCV – Untuk manipulasi gambar dan video
  • 🌐 Flask – Framework Python ringan untuk backend web
  • 💻 Bootstrap 5 – Framework frontend untuk UI responsif
  • 📄 CSV Export – Menyimpan hasil klasifikasi untuk dokumentasi

🧪 Cara Kerja Aplikasi

  1. 👁️ Pengguna membuka kamera atau mengunggah gambar tomat
  2. 🧠 Model YOLOv8 mendeteksi tomat dan menandai tingkat kematangan
  3. 📊 Sistem menampilkan hasil klasifikasi lengkap beserta deskripsinya
  4. 💾 Pengguna dapat mengunduh hasil deteksi dalam format .csv

⚙️ Fitur Unggulan

📸 Kamera Real-Time

Gunakan kamera perangkat untuk mendeteksi tomat secara langsung dengan bounding box dan label kematangan.

🖼️ Upload Gambar

Unggah gambar dari galeri untuk diklasifikasikan secara otomatis oleh sistem.

📥 Ekspor CSV

Unduh hasil deteksi lengkap dengan label dan deskripsi dalam format .csv.

📊 Statistik Deteksi

Tampilkan grafik jumlah deteksi per kategori secara otomatis.

👨‍💻 Tentang Pengembang

Aplikasi ini dikembangkan oleh Ibrahim Ofick sebagai bagian dari proyek pembelajaran dan implementasi sistem Computer Vision. Fokus utama adalah pemanfaatan teknologi AI untuk pertanian presisi dan efisiensi proses panen.

Proyek ini juga dirancang agar dapat diintegrasikan lebih lanjut ke sistem pertanian berbasis IoT, seperti kamera otomatis, drone pengawas lahan, atau detektor kualitas panen berbasis sensor visual.
